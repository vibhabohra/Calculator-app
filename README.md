# Calculator App 🧮

A simple and elegant calculator web application built with HTML, CSS, and JavaScript. Perfect for basic arithmetic operations with a clean, user-friendly interface.

## Features ✨

- **Basic Operations**: Addition, subtraction, multiplication, and division
- **Clean Interface**: Modern, intuitive design with clear buttons and display
- **Real-time Calculation**: Instant results as you enter values
- **Error Handling**: Prevents invalid operations
- **Responsive Design**: Works on different screen sizes

## Demo 🎯

The calculator provides a straightforward way to perform basic mathematical calculations. Simply click the number buttons, select your operation, and press equals for the result.

## Technologies Used 💻

- **HTML5** - Structure
- **CSS3** - Styling and layout
- **JavaScript** - Functionality and calculations

## Getting Started 🚀

1. **Clone the repository**
```bash
   git clone https://github.com/vibhabohra/Calculator-app.git
```

2. **Open the application**
   - Navigate to the project folder
   - Open `index.html` in your web browser

3. **Use the calculator**
   - Enter numbers using the numeric buttons
   - Select an operation (+, -, ×, ÷)
   - Click "=" to see the result
   - Click "C" to clear

## How to Use 📖

| Button | Function |
|--------|----------|
| 0-9 | Enter numbers |
| +, -, ×, ÷ | Select operation |
| = | Calculate result |
| C | Clear display |
